When is a fd ready?

As seen, IO Multiplexing depends on kernel informing about readiness to process.

Readiness depends on type of file

sockets are ready when input/output avialable.

sockets are ready when an incoming connection established on listening socket .

pipe's read end is ready if atleast one byte is there to read irrespective of length requested.

Disk files are always marked ready. So these don't work.
